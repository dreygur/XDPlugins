# AdobeXD-Count-Text
An Adobe XD plugin for counting characters and words in text layers.

## Usage

1. Select one or more text layers
2. Select `Plugins->Count Text` or use keyboard shortcut `Cmd+Alt+C`(Mac) or `Ctrl+Alt+C`(Windows)
